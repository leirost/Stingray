# Specification


## Types:
Types can be optional and non-optional. Language supports Arrays.

- Integer
- Rational
- String
- Boolean
- User-Defined
    - Classes (and Interfaces)
    - Functions
    

## Operations:
+ Hash-code (#)
- Relational (=, !=, <, >, <=, >=)
    + Can override =, < and <=. [!= is not =, > is not <=, >= is not <]
+ Arithmetic binary (+, -, *, /, %, ^) and unary (+, -)
- Logical binary (and, xor, or, =>, <=>) and unary (not)
- Pattern-Matching: is
